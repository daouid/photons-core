class photons_colour.Parser¶

Class for converting valid colour specifiers into the options for a SetWaveformOptional device command.

Note

http://colorizer.org/ and http://colormine.org/ are amazing!

The valid colour names are:

'blue' 'cyan' 'green' 'orange' 'pink' 'purple' 'red' 'white' 'yellow'

And valid colour specifiers are as follows:

01_hsb_component
\Ahsb:([\d\.]+),([\d\.]+%?),([\d\.]+%?)\Z
02_rgb_component
\Argb:(\d+,\d+,\d+)\Z
03_hex_component
\A(?:hex:)?#?([0-9a-fA-F]{6})\Z
04_hue_component
\Ahue:([\d.]+)\Z
05_saturation_component
\Asaturation:([\d.]+%?)\Z
06_brightness_component
\Abrightness:([\d.]+%?)\Z
07_kelvin_component
\Akelvin:([\d]+)\Z
10_random_component
random
whitespace
\s+

This class also has some classmethods for your use:
